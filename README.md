# Le Baobab Gourmand


Site vitrine pour le restaurant "Le Baobab Gourmand", présentant une expérience culinaire africaine authentique à Dakar.

## Description

Ce projet est un site web statique construit en HTML, CSS et JavaScript. Il met en avant le restaurant, son menu, son histoire et permet aux visiteurs de contacter le restaurant pour une réservation ou une commande.

## Pages du site

- `index.html` : page d'accueil avec une présentation du restaurant, des plats signature, des plats vedettes, des témoignages et des chiffres clés.
- `menu.html` : page du menu avec plusieurs catégories de plats (entrées, plats principaux, desserts, boissons) et les boutons filtres par categories .
- `about.html` : page "À propos" qui explique l'histoire du restaurant, ses valeurs et présente l'équipe.
- `contact.html` : page de contact et réservation avec formulaire et carte Google Maps.
- `Banniere.html` : page de navigation ou de bannière utilisée comme lien de retour en bas de page.

## Fichiers principaux

- `style.css` : styles principaux du site.
- `script.js` : comportement JavaScript du site (scroll, formulaire, interaction, etc.).
- `image/` : dossier contenant les images utilisées pour les plats, l'équipe et le restaurant.

## Fonctionnalités

- **Navbar responsive** : Navigation adaptative sur tous les appareils
- **Carousel d'images** : Présentation visuelle des plats (si implémenté)
- **Menu interactif** : Affichage des plats avec images, descriptions et prix en FCFA
- **Formulaire** : Permet aux clients de réserver une table ou commander
- **Carte interactive** : Intégration Google Maps pour la localisation
- **Design élégant** : Inspiré de la cuisine africaine avec des polices et couleurs appropriées
